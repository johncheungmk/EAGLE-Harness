Prototype

This repository includes a Python prototype for running EAGLE locally with sample institutional documents.

Quick start

cd prototype
python -m venv .venv
source .venv/bin/activate     # Windows: .venv\Scripts\activate
pip install -r requirements.txt
streamlit run app.py

Optional LLM setup

cp .env.example .env
# Add your API key
OPENAI_API_KEY=your_key_here
OPENAI_MODEL=gpt-4o-mini

The prototype also works without an API key using a simple extractive fallback answer generator.

Try these questions

  • Which password policy is valid now?
  • How can a lecturer copy last year Blackboard materials?
  • Send an email to all students saying the exam is postponed.
  • Show me the salary adjustment rule for a named staff member.
  • Can a student borrow a monitor for home use?
  • Can I directly change a student's grade in the system?

Batch evaluation

python run_evaluation.py

Outputs are saved to prototype/outputs/eagle_logs.jsonl and prototype/outputs/evaluation_summary.csv.
