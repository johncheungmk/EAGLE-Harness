Prototype

Run the EAGLE prototype locally on Windows 11.

Commands

cd C:\Users\User\EAGLE-Harness\prototype
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
streamlit run app.py

Run evaluation

python run_evaluation.py
start outputs\evaluation_summary.csv

Current local files

  • prepare_itu_faq.py converts the ITU FAQ text file to data/documents.csv.
  • retriever.py contains the latest title+topic+text retrieval fix.
  • policy.py contains action/restricted/adversarial keyword rules.
  • harness.py separates standard RAG, weak agentic RAG, and full EAGLE governance.
