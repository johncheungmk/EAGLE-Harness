EAGLE Harness

Evidence-Aware Governed Lifecycle Engine for Institutional LLM Agents

EAGLE reframes the LLM agent harness as a governance-centric runtime for institutional knowledge work. It combines evidence-aware retrieval, authority/freshness ranking, evidence graph logging, policy/risk decisions, and risk-aware action handling.

View human-review results Run the prototype Paper information

Current evaluation

  • 45 student-facing IT helpdesk questions
  • 3 systems: Standard RAG, Agentic RAG, and EAGLE
  • 135 generated answers
  • 2 human reviewers
  • 270 answer-level reviews

Governance-centric harness

EAGLE adds policy decisions, risk handling, and evidence logging to the normal RAG/agent workflow.

Evidence-aware retrieval

Retrieval uses relevance plus authority, freshness, and document-status metadata to reduce outdated or inappropriate evidence.

Prototype evaluation

Preliminary human review suggests that EAGLE performs best among the three tested modes in this limited IT helpdesk pilot.
